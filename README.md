# Databel Churn Analysis-Dashboard
# 1. Project Title / Headline
📊 Databel Churn Analysis: Databel Customer Churn Insights Dashboard
A dynamic, interactive data visualization tool to explore customer analysis data- focusing on churn rate, customer demographics, service usage patterns, and revenue impact.
# 2. Short Description / Purpose
The Databel Churn Analysis Dashboard is a visually engaging and analytical Power BI report designed to help users to quickly identify customers who are likely to churn and understand the reasons behind their behavior. It enables users to monitor key metrics like churn rate, customer segments, and trends in one place. By providing clear and interactive insights, it supports better decision-making and helps businesses take proactive actions to improve customer retention and increase revenue. 
# 3. Tech Stack
The dashboard was built using the following tools and technologies:
• 📊 Power BI Desktop – Main data visualization platform used for report creation.
• 📂 Power Query – Data transformation and cleaning layer for reshaping and preparing the data.
• 🧠 DAX (Data Analysis Expressions) – Used for calculated measures, dynamic visuals, and conditional logic.
• 📝 Data Modeling – Relationships established among tables (Churn Label, Churn Category, Churn Reason) to enable cross-filtering and aggregation.
• 📁 File Format – .pbix for development and .png for dashboard previews.
# 4. Data Source
The data for this dashboard comes from the Databel dataset, which includes customer information, service usage details, and account-related metrics. It typically contains fields such as customer ID, demographics (age, location), subscription details, usage patterns, billing information, and churn status.

The data is structured in a tabular format, where each row represents an individual customer and each column represents a specific attribute or variable. This structured format makes it easy to clean, analyze, and visualize the data to extract meaningful insights about customer churn.
# 5. Features / Highlights
🔹 Business Problem
Databel was facing a significant increase in customer churn, leading to revenue loss and reduced customer lifetime value. The company lacked clear visibility into the reasons behind customer attrition, making it difficult to take proactive actions. There was a need to analyze customer data to uncover patterns, trends, and key factors contributing to churn.

🔹 Goal of the Dashboard
The main goal of this dashboard is to provide a comprehensive view of customer churn by analyzing various factors such as demographics, service usage, contract type, and billing information. It aims to help stakeholders easily track churn metrics, identify high-risk customers, and support data-driven decision-making to improve retention strategies.

🔹 Key Visuals (Walkthrough)
The dashboard includes:
KPI Cards displaying key metrics such as total customers, churn rate, and active vs churned customers
Churn by Demographics (age group, gender, location) to identify which customer segments are more likely to leave
Service Usage Analysis showing how product/service engagement impacts churn
Contract Type & Payment Method Analysis to understand how subscription models influence customer retention
Monthly Churn Trend line chart to track churn patterns over time
Interactive Filters/Slicers allowing users to drill down into specific customer segments

🔹 Business Impact & Insights
The dashboard revealed key insights such as higher churn among specific customer segments and contract types, along with patterns linked to service usage and billing. These insights help the business:
Identify and target high-risk customers early
Improve pricing and service strategies
Enhance customer experience and engagement
Reduce churn rate and increase customer retention
Support strategic, data-driven business decisions
